a game where you guess a random number

## How to play

enter this link -> https://numbers-game.thsite.top/

1. enter your guess
2. if the secret number is lower it will say "too high"
 and if the secret number is higher it will say "too low"
3. if you guess the number correctly it will say "Congratulations You won in x attempts"
4. your best score is your the least number of attempts you used to guess the number
5. click "New Game" to play again

## Technology used

this project is made using php, html, css and deployed on infinityfree

it uses session data in php and post requests in html to send to php

html is embedded in the php file
